ðŸ“Š How It Works

1

Data Collection

Users provide information about demographics, lifestyle, mental health indicators, and substance use through a comprehensive questionnaire.

2

Feature Processing

The system processes 16 different features including sleep hours, physical activity, stress levels, and more.

3

ML Prediction

Four different machine learning models analyze the data simultaneously to predict risk levels: Low, Medium, or High.

4

Results & Recommendations

Users receive personalized suggestions and can view detailed model comparisons and performance metrics.

ðŸ“š Dataset

The model was trained on a synthetic dataset of 1,000 samples with balanced distribution across risk levels:

  • Low Risk: 35%
  • Medium Risk: 40%
  • High Risk: 25%

The dataset includes 16 features carefully selected based on mental health research and clinical practices.

ðŸ”¬ Model Performance

All models were evaluated using standard machine learning metrics:

  • Accuracy: Overall correctness of predictions
  • Precision: Accuracy of positive predictions for each class
  • Confusion Matrix: Detailed breakdown of predictions vs actual values

Visit the Model Analysis page for detailed performance metrics and comparisons.
